# Brainwave - Modern AI Landing Page Clone

A high-fidelity, responsive landing page built with **React.js** and **Tailwind CSS**. This project showcases complex layouts, sleek dark-mode aesthetics, and modern UI interactions.

## 🚀 Live Demo
**Check out the live site:** [naser-brainwave.vercel.app](https://naser-brainwave.vercel.app/)

---

## ✨ Features
* **Modern UI/UX:** Sophisticated "Bento Box" grid layouts and futuristic aesthetic.
* **Fully Responsive:** Optimized for mobile, tablet, and desktop viewing.
* **Parallax & Animations:** Smooth scrolling effects and interactive SVG elements.
* **Production Ready:** Successfully deployed and optimized for **Vercel**.

## 🛠️ Tech Stack
* **Frontend:** React.js
* **Styling:** Tailwind CSS
* **Build Tool:** Vite
* **Deployment:** Vercel

---

## 🏗️ Local Development

To get a local copy up and running, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Smook-e/brainwave-clone.git
   ```
   # Enter project directory
   ```bash
   cd BrainWave
   ```
   
   # Install dependencies
   ```bash
   npm install
   ```
   
3. **Development**
   # Start dev server
   ```bash
   npm run dev
   ```
   
   # Open http://localhost:5173 in your browser.
